⚠️ Known Issue

There is an ongoing issue in the hosted application in the /track_task endpoint, where the API endpoint is getting redirected to HTTP from HTTPS. Most modern web browsers do not support the redirection, so the API endpoint is not accessible. I have tried adding app-level middleware to solve this issue where I am enforcing HTTPS redirect, but it too is not solving the issue. I will fix this issue in the future. But for now, you can access this API from the Swagger UI. I regret the inconvenience.
